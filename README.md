# 🍔 Burger Co.

A premium, high-performance landing page for a fictional gourmet burger restaurant, built with modern web technologies. The project demonstrates a stunning visual identity with a dark, crimson-and-gold theme, smooth micro-interactions, and beautiful typography.

## ✨ Features

- **Premium Design:** Curated, visually stunning dark mode theme with cohesive typography and gold/crimson accents.
- **Fluid Animations:** Smooth scrolling effects, reveal animations, and micro-interactions powered by Framer Motion.
- **Dynamic Menu:** A beautifully curated section showcasing the restaurant's signature burgers.
- **Fully Responsive:** Carefully crafted layouts for all devices, from mobile phones to ultra-wide displays.
- **Modern Tech Stack:** Built with Next.js App Router for optimal performance, server-side rendering, and easy deployment.

## 🛠️ Tech Stack

- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

Follow these steps to run the application locally.

### 1. Clone the repository

```bash
git clone <repository-url>
cd burger-co
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `/app`: The main Next.js App Router directory containing pages and global layouts.
- `/components`: Reusable React components (UI elements, layout sections).
- `/lib`: Helper functions, utilities, and static mock data (e.g., menu configurations).
- `/public`: Static assets like images and fonts.
